# Eco_Pack_AI
EcoPack AI is an innovative web application designed to help businesses make eco-friendly and cost-effective packaging decisions. Using advanced machine learning models, it predicts both the carbon footprint and the financial cost of different packaging options, empowering companies to choose solutions that are sustainable and economical.
